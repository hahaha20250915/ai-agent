# Java 编程学习路线（示例版）

## 第一阶段：语言基础

- Java 语法、流程控制、方法、数组
- 面向对象：类与对象、继承、多态、接口、抽象类

## 第二阶段：核心进阶

- 集合框架、异常处理、IO/NIO
- 多线程与并发基础
- JVM 基础：内存结构、垃圾回收的基本概念

## 第三阶段：Web 与框架

- Servlet / Spring / Spring MVC 基本概念
- Spring Boot 快速构建 Web 项目
- 使用 MySQL + MyBatis / JPA 进行数据持久化

## 第四阶段：实战与项目

- 至少完整做 1–2 个中小型项目，覆盖登录、权限、CRUD、日志等常见场景。
- 对项目做总结：技术选型、架构设计、遇到的问题和优化点。
