# 程序员常见面试题（示例版）

> 你可以在这个文件里补充、维护自己的面试题和思路。

## Java 基础

- 面向对象三大特性是什么？如何在项目中体现？
- `ArrayList` 和 `LinkedList` 的区别？适用场景？
- `==` 和 `equals` 的区别？
- `HashMap` 的工作原理？为什么要重写 `hashCode` 和 `equals`？

## 并发与多线程

- 创建线程的几种方式？优缺点？
- 线程池的核心参数有哪些？如何合理配置？
- `synchronized` 与 `ReentrantLock` 的区别？

## Spring / Spring Boot

- `@Component`、`@Service`、`@Repository` 有什么区别？
- Bean 的生命周期大致包括哪些阶段？
- 什么是 AOP？举一个在项目中使用的例子。

## 数据库

- 事务的四大特性（ACID）分别是什么？
- 索引的优点和缺点？什么情况下不适合建索引？
